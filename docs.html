Click on the button below to go to the access token generator page:

Generate Access Token

Step 2: Grab the Token and Access API Endpoints

Once you have generated the access token, you can use it to access Our API endpoint

How to Access Our APIs

Step 3: How to Use Our APIs

To use our APIs, follow these steps:

1. Add User to Database Endpoint :

This endpoint allows you to add a user to the database.

It requires the following parameters:

  • bot_token: The actual Bot token of your telegram Bot.
  • user_id: The telegram id of your Bot user
  • Access Token : The Access Token Genarated By our service..

Here's an example of how to make a request using Js (Bot.Buisness):

const url = "https://addteleservices.soumya765.workers.dev"

  HTTP.post({
    url: url,
    headers: { "content-type": "application/json" },
        body: {
          access_token: "Your Access Token",
          bot_token: "Your Bot Token",
          user_id: "User telegram Id"
        },
  })

//You can Use both Post or Get Method

2. Broadcast Message Endpoint (POST /api/broadcast):

This endpoint allows you to broadcast a message to multiple users.

It requires the following parameters:

  • bot_token: The actual Bot token of your telegram Bot.
  • admin_id: The Admin id of your Bot
  • access_token : The Access Token Genarated By our service..
  • caption: Yor caption t.
  • text: The telegram id of your Bot user
  • file_id:: The file like photo, video etc ... , That you want to Broadcast .
  • text: The message you want to broadcast.

Here's an example of how to make a request using JS(Bot.Buisness):

var url = "https://broadcast.soumya765.workers.dev"
          
      HTTP.post({
        url: url,
        headers: { "content-type": "application/json" },

          access_token: "Your Access Token ",
